# rust todo app

Rust todo app user rspc and supabase auth

## Features

- Login
- Register
- CRUD todo

## Tech Stack

### Frontend

- Reactjs
- rspc
- Typescript
- Tailwind css
- Supabase auth

### Backend

- rspc
- axum
- sqlx

## Usage

```bash
# Terminal One
cd web
yarn
yarn dev

# Terminal Two
cargo run
```
